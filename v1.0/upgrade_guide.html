会员升级指南

适用于 Portal 内的会员升级流程（可自行替换为图文说明）。

步骤 1：打开升级面板

1
点击右上角账号菜单里的“升级会员”按钮，弹出升级面板。
（这里放截图：右上角菜单 + 升级按钮）

步骤 2：选择会员等级与网络

2
选择 Pro 或 Ultra，推荐使用 Ethereum 网络（标注为“推荐”）。
（这里放截图：Plan / Network 选择区域）

步骤 3：转账并复制交易 Hash

3
使用钱包向指定地址转账对应金额，完成后复制交易 Hash。
（这里放截图：钱包转账界面 + 交易哈希）

步骤 4：提交验证

4
在输入框粘贴交易 Hash，点击“提交”，等待验证完成。
（这里放截图：Tx Hash 输入框 + 提交按钮）
注意：Solana 暂不支持，请勿在 Solana 网络充值。

常见问题

Q
验证失败怎么办？
请确认交易 Hash 正确且转账网络与选择的网络一致。如仍失败，请联系管理员。
